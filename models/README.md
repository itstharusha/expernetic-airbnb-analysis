# Model Artifacts

This directory stores model artifacts generated by the machine learning pipeline. 

Artifacts include:
- `xgboost_model.joblib`: The trained XGBoost model object.
- `shap_explainer.joblib`: The fitted SHAP explainer for the model.
- `model_meta.joblib`: Metadata containing features, encoding mappings, and imputation medians.

All model artifacts are generated reproducibly by `scripts/train_model.py`. Experiment configurations and metrics are tracked in the `experiments/` directory.
